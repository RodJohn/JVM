
# JVM内存

概述

    JVM Runtime Data Area（运行时数据区）
    也叫作JVM内存
    
作用

    类加载器加载java字节码文件，执行引擎程序所用的就是JVM内存
    
# 组成

    JVM内存由
    线程独享的
        PC计数器，java栈，本地方法区
    线程共享的        
        java堆，方法区组成
