

# 日志

-XX:+PrintGCDetails 

	打印日志详情

-XX:+PrintGCTimeStamps

	打印日志时间

-XX:PrintHeapAtGC: 

	打印GC前后的详细堆栈信息

-Xloggc:filename:

	把相关日志信息记录到文件
	
	
# 内存溢出导出


-XX:-OutOfMemoryError

	OOM时执行脚本

-XX:-HeapDumpOnOutOfMemoryError

	OOM时导出堆到文件
	
-XX:HeapDumpPath

	导出堆存放地址


# 其他

-XX:+DisableExplicitGC

	禁止代码中显示调用GC
